Using object detection for data collection
Hello and welcome to a tutorial on how to use object detection for data collection. Here are the prerequisites before you start. First you should have opened the website roboflow which is linked to the left. Next you will create an account for the website for this you will need to use a personal google account. During setup you will select personal use when prompted. Then when prompted to set up you will create an object detection project and add a test label to the project. This is all for the start now you can head to step one.
