Step 1

The first step is to find images for the object detection model. For this we will be using google dataset search. This tool will allow you to find images for identification of many different objects. For this example let's use dogs and cats. Some datasets that you might want to use are the stanford dog dataset and the oxford pet dataset for cats. I would recommend using 1000 pictures for each dog and cat as it will allow the images to be nicely spread between the train, validate and test datasets. In the next step you will be marking the images and applying transformations. Refer to step 2 for the next part of the lesson.
