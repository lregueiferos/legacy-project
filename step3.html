Step 3

Once you have completed step 1 and 2 you can now start the generation of the model. To do this you will first need to apply transformations to the images. I would recommend you apply a grayscale filter and a crop filter to make the model more effective and achieve better results. And for augmentations I would recommend a blur, cutout and flip augmentation. These will give you good results for most types of images but you may need to adjust these if you are using images that need to identify small details in the images. For such use cases use no augmentations and a high resolution image for the dataset. Then you can start the creation of the model. This may take up to a day so start this as soon as possible and monitor it as it may fail and need to be redone. Now if you would like to see an example of expected results of the training then go to the example section of the site.
